# ScissorsButtons

#### Scissors & Button 게임 ->  가로, 세로, 대각선에 같은 그림 모양을 없애 모두 5X5를 없애는 게임

java swing audio

기능

* 가로 세로 대각선 삭제
* 중간에 다른 음식이 존재 시 삭제 불가능
* 모두 삭제 시 점수, 횟수, 시간을 출력
* 게임 도중 도망 기능
* 한번의 되돌리기 기능
* audio wav파일 재생
